# Marketing Advisor

Claude Code용 웹서비스 마케팅 전략 수립 스킬입니다. 대화형 질문을 통해 서비스 정보를 수집하고, 웹 리서치를 기반으로 맞춤형 마케팅 플랜을 제공합니다.

## 주요 기능

- **서비스 분석**: URL 기반 자동 분석 + 대화형 질문으로 정보 수집
- **경쟁사 리서치**: 웹 검색으로 경쟁사 및 시장 트렌드 조사
- **채널 전략**: 서비스 유형/예산에 맞는 마케팅 채널 추천
- **도구 추천**: 예산별 마케팅 도구 스택 제안
- **실행 계획**: 30일/90일 액션 플랜이 포함된 마케팅 플랜 문서 생성

## 설치

### 1. 저장소 클론

```bash
git clone https://github.com/greeun/marketing-advisor.git
```

### 2. Claude Code 스킬 디렉토리에 심볼릭 링크 생성

```bash
ln -s $(pwd)/marketing-advisor ~/.claude/skills/marketing-advisor
```

### 3. Claude Code 재시작

새 세션을 시작하면 스킬이 자동으로 로드됩니다.

## 사용법

### 트리거 키워드

다음 키워드로 스킬을 활성화할 수 있습니다:

- `마케팅 전략`
- `마케팅 분석`
- `그로스 전략`
- `마케팅 플랜`
- `경쟁사 분석`
- `마케팅 도구 추천`
- `런칭 마케팅`
- `growth strategy`
- `marketing plan`

### 예시

```
사용자: 우리 서비스 마케팅 전략 좀 세워줘

Claude: 마케팅 전략을 수립하겠습니다. 먼저 몇 가지 질문드릴게요.
        1. 서비스 URL이 있으신가요?
        2. 서비스를 한 문장으로 설명해주시겠어요?
        ...
```

```
사용자: https://example.com 마케팅 플랜 만들어줘. B2B SaaS고 예산은 월 100만원이야.

Claude: [URL 분석 → 경쟁사 리서치 → 채널 전략 → 도구 추천 → 마케팅 플랜 문서 생성]
```

## 워크플로우

```
Phase 1: 서비스 이해
    ↓ 대화형 질문 + URL 분석
Phase 2: 시장 리서치
    ↓ 경쟁사 분석, 트렌드, 키워드
Phase 3: 채널 전략 수립
    ↓ 서비스 유형/예산별 채널 추천
Phase 4: 도구 추천
    ↓ 예산별 마케팅 도구 스택
Phase 5: 마케팅 플랜 문서 생성
    → 30일/90일 실행 계획 포함
```

## 지원 서비스 유형

| 유형 | 예시 | 주요 전략 방향 |
|------|------|---------------|
| SaaS/B2B | 업무 도구, 분석 서비스 | 콘텐츠 마케팅, LinkedIn, 웨비나 |
| B2C 앱 | 모바일 앱, 소비자 서비스 | ASO, SNS, 인플루언서 |
| 이커머스 | 온라인 쇼핑몰 | 쇼핑 광고, SEO, 리타겟팅 |
| 플랫폼 | 마켓플레이스, 커뮤니티 | 커뮤니티, 바이럴, 네트워크 효과 |
| 콘텐츠 | 미디어, 뉴스레터 | SEO, 뉴스레터, SNS |

## 예산별 도구 스택 예시

### 부트스트래핑 (월 0-50만원)
- 분석: GA4 + Hotjar 무료
- 이메일: Mailchimp/Brevo 무료
- 소셜: Buffer 무료
- 자동화: n8n

### 스타트업 (월 50-200만원)
- 분석: GA4 + Amplitude
- SEO: Ahrefs Lite
- 이메일: ConvertKit
- 자동화: Zapier

### 성장기 (월 200만원+)
- 분석: Mixpanel
- SEO: Ahrefs Standard
- 이메일: Klaviyo
- CRM: HubSpot

## 출력 결과물

스킬 실행 완료 시 다음 내용이 포함된 마케팅 플랜 문서가 생성됩니다:

1. **Executive Summary** - 서비스 개요, 타겟 고객, 경쟁 우위
2. **시장 분석** - 경쟁사 분석, 트렌드, 기회/위협
3. **마케팅 전략** - 포지셔닝, 채널 전략, 콘텐츠 전략
4. **실행 계획** - 30일 액션 플랜, 90일 마일스톤
5. **도구 및 리소스** - 추천 도구 스택, 예상 비용
6. **측정 및 최적화** - 핵심 KPI, 리뷰 주기
