# Marketing Advisor

웹서비스의 마케팅 전략을 수립하는 대화형 컨설팅 스킬. 서비스 분석부터 실행 가능한 마케팅 플랜까지 단계별로 가이드합니다.

## Workflow

```
Phase 1: 서비스 이해
    ↓
Phase 2: 시장 리서치
    ↓
Phase 3: 채널 전략 수립
    ↓
Phase 4: 도구 추천
    ↓
Phase 5: 마케팅 플랜 문서 생성
```

---

## Phase 1: 서비스 이해

### 1.1 기본 정보 수집

AskUserQuestion 도구로 아래 정보를 수집:

| 질문 | 목적 |
|------|------|
| 서비스 URL이 있나요? | 웹사이트 분석용 |
| 서비스를 한 문장으로 설명해주세요 | 핵심 가치 파악 |
| 타겟 고객은 누구인가요? | 페르소나 정의 |
| 현재 단계는? (아이디어/MVP/운영중/성장기) | 전략 방향 결정 |
| 월 마케팅 예산은? | 도구/채널 추천 기준 |

### 1.2 URL 분석 (URL 제공 시)

WebFetch로 서비스 페이지 분석:

```
분석 항목:
- 메인 카피/슬로건
- 핵심 기능 및 특징
- 가격 정책
- CTA (Call to Action)
- 소셜 프루프 (고객 후기, 사용 기업 등)
```

### 1.3 서비스 유형 분류

| 유형 | 특징 | 주요 전략 방향 |
|------|------|---------------|
| **SaaS/B2B** | 기업 고객, 구독 모델 | 콘텐츠 마케팅, LinkedIn, 웨비나 |
| **B2C 앱** | 일반 소비자, 앱스토어 | ASO, SNS, 인플루언서 |
| **이커머스** | 상품 판매 | 쇼핑 광고, SEO, 리타겟팅 |
| **플랫폼** | 양면 시장 | 커뮤니티, 바이럴, 네트워크 효과 |
| **콘텐츠** | 정보/미디어 | SEO, 뉴스레터, SNS |

## Phase 2: 시장 리서치

### 2.1 경쟁사 분석

WebSearch로 경쟁사 조사:

```
검색어 예시:
- "[서비스 키워드] alternatives"
- "[서비스 키워드] vs"
- "best [서비스 카테고리] tools 2025"
```

**분석 프레임워크:**

| 항목 | 우리 서비스 | 경쟁사 A | 경쟁사 B |
|------|------------|---------|---------|
| 핵심 가치 | | | |
| 가격 | | | |
| 타겟 고객 | | | |
| 강점 | | | |
| 약점 | | | |

### 2.2 시장 트렌드

WebSearch로 트렌드 조사:

```
검색어:
- "[산업] market trends 2025"
- "[카테고리] growth statistics"
- "[타겟 고객] behavior trends"
```

### 2.3 키워드 리서치

SEO/콘텐츠 전략을 위한 키워드 도출:

- **핵심 키워드**: 서비스 직접 관련
- **문제 키워드**: 타겟 고객의 문제/니즈
- **비교 키워드**: "A vs B", "best X"
- **정보 키워드**: "how to", "what is"

## Phase 3: 채널 전략 수립

### 3.1 채널 매트릭스

서비스 유형과 예산에 따라 채널 우선순위 추천:

| 채널 | 비용 | 시간 | SaaS | B2C | 이커머스 |
|------|------|------|------|-----|---------|
| SEO/블로그 | 낮음 | 장기 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| 소셜미디어 | 낮음 | 중기 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 유료 광고 | 높음 | 단기 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 이메일 | 낮음 | 중기 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ |
| 커뮤니티 | 낮음 | 장기 | ⭐⭐⭐ | ⭐⭐ | ⭐ |
| PR/미디어 | 중간 | 단기 | ⭐⭐ | ⭐⭐⭐ | ⭐⭐ |
| 인플루언서 | 중간 | 단기 | ⭐ | ⭐⭐⭐ | ⭐⭐⭐ |
| 파트너십 | 낮음 | 장기 | ⭐⭐⭐ | ⭐⭐ | ⭐⭐ |

### 3.2 단계별 전략

**런칭 단계 (0→1)**
1. 핵심 채널 1-2개 집중
2. 초기 사용자 확보 (Product Hunt, 커뮤니티)
3. 피드백 루프 구축

**그로스 단계 (1→N)**
1. 유효 채널 스케일업
2. 유료 광고 테스트
3. 리텐션/리퍼럴 최적화

## Phase 4: 도구 추천

### 4.1 도구 카테고리

| 카테고리 | 무료/저가 | 유료 |
|----------|----------|------|
| **분석** | GA4, Hotjar(무료) | Mixpanel, Amplitude |
| **SEO** | Google Search Console, Ubersuggest | Ahrefs, SEMrush |
| **이메일** | Mailchimp(무료), Brevo | ConvertKit, Klaviyo |
| **소셜** | Buffer(무료), Later | Hootsuite, Sprout Social |
| **광고** | Google/Meta 직접 | AdEspresso |
| **CRM** | HubSpot(무료), Notion | Salesforce, Pipedrive |
| **자동화** | n8n, Zapier(무료) | Make, Zapier Pro |
| **랜딩페이지** | Carrd, Notion | Webflow, Framer |

### 4.2 예산별 스택 추천

**부트스트래핑 (월 0-50만원)**
```
분석: GA4 + Hotjar 무료
이메일: Mailchimp/Brevo 무료
소셜: Buffer 무료
랜딩: Carrd ($19/년)
자동화: n8n (무료)
```

**스타트업 (월 50-200만원)**
```
분석: GA4 + Amplitude 스타터
SEO: Ahrefs Lite
이메일: ConvertKit Creator
광고: Google/Meta 직접 관리
자동화: Zapier Starter
```

**성장기 (월 200만원+)**
```
분석: Mixpanel Growth
SEO: Ahrefs Standard
이메일: Klaviyo
CRM: HubSpot Marketing
자동화: Make/n8n + 전문 도구
```

## Phase 5: 마케팅 플랜 문서 생성

모든 분석 완료 후, 마케팅 플랜 문서를 생성합니다.

### 문서 구조

```markdown
# [서비스명] 마케팅 플랜

## 1. Executive Summary
- 서비스 개요
- 핵심 타겟 고객
- 주요 경쟁 우위

## 2. 시장 분석
- 경쟁사 분석 요약
- 시장 트렌드
- 기회 및 위협

## 3. 마케팅 전략
### 3.1 포지셔닝
- 핵심 메시지
- 차별화 포인트

### 3.2 채널 전략
| 채널 | 목표 | KPI | 예산 |
|------|------|-----|------|
| ... | ... | ... | ... |

### 3.3 콘텐츠 전략
- 콘텐츠 유형
- 주요 토픽/키워드

## 4. 실행 계획
### 4.1 30일 액션 플랜
| 주차 | 액션 아이템 | 담당 | 완료 기준 |
|------|------------|------|----------|
| 1주 | ... | | |
| 2주 | ... | | |
| 3주 | ... | | |
| 4주 | ... | | |

### 4.2 90일 마일스톤
- Month 1: ...
- Month 2: ...
- Month 3: ...

## 5. 도구 및 리소스
- 추천 도구 스택
- 예상 비용

## 6. 측정 및 최적화
- 핵심 KPI
- 리뷰 주기
```
