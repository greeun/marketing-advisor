## 2. Google Ads

### 캠페인 유형
| 유형 | 목적 | 추천 상황 |
|------|------|----------|
| Search | 검색 의도 타겟 | 전환 목표, 바텀 퍼널 |
| Performance Max | 자동 최적화 | 다양한 채널 통합 |
| Display | 리마케팅, 인지도 | 브랜딩, 리타겟팅 |
| YouTube | 영상 광고 | 인지도, 제품 데모 |

### 검색 광고 최적화
1. **키워드 구조**
   - 브랜드 키워드: 최우선
   - 제품/서비스 키워드: 높은 전환 의도
   - 경쟁사 키워드: 주의 필요
   - 일반 키워드: 넓은 도달

2. **품질 점수 향상**
   - 광고 관련성: 키워드-광고 일치
   - 예상 CTR: 매력적인 카피
   - 랜딩 페이지 경험: 빠른 로딩, 관련성

3. **입찰 전략**
   - 시작: 수동 CPC로 데이터 수집
   - 성숙: 타겟 CPA/ROAS 자동 입찰

## 3. Meta Ads

### 캠페인 목표
| 목표 | 최적화 대상 | 추천 상황 |
|------|------------|----------|
| 인지도 | 도달, 노출 | 브랜드 론칭 |
| 트래픽 | 링크 클릭 | 콘텐츠 마케팅 |
| 전환 | 구매, 리드 | 퍼포먼스 마케팅 |
| 앱 설치 | 설치 수 | 앱 프로모션 |

### 타겟팅 전략
1. **코어 오디언스**: 관심사, 행동, 인구통계
2. **커스텀 오디언스**: 웹사이트 방문자, 고객 목록
3. **유사 오디언스**: 기존 고객과 유사한 사용자

### 크리에이티브 베스트 프랙티스
- **이미지**: 텍스트 20% 미만, 밝고 눈에 띄는 색상
- **영상**: 첫 3초에 핵심, 무음 재생 고려
- **카피**: 짧고 명확한 CTA, 이모지 활용
- **포맷**: 피드, 스토리, 릴스 각각 최적화

## 4. 예산 배분

### 단계별 전략

**테스트 단계 (1-2개월)**
- 총 예산의 20-30%를 테스트에 사용
- 여러 오디언스, 크리에이티브 A/B 테스트
- CPA/ROAS 데이터 수집

**최적화 단계 (3-6개월)**
- 성과 좋은 캠페인에 예산 집중
- 타겟 세분화
- 크리에이티브 리프레시

**스케일 단계 (6개월+)**
- 검증된 캠페인 확대
- 새로운 채널 테스트
- 시즌/이벤트 대응

### 예산 배분 예시 (월 100만원)
```
Google Search: 40만원 (40%)
- 브랜드 키워드: 10만원
- 전환 키워드: 30만원

Meta Ads: 50만원 (50%)
- 리타겟팅: 15만원
- 관심사 타겟팅: 25만원
- 유사 오디언스: 10만원

테스트 버퍼: 10만원 (10%)
```

## 5. 측정 및 최적화

### 핵심 지표
| 지표 | 공식 | 좋은 기준 |
|------|------|----------|
| CTR | 클릭/노출 | 검색 2%+, 디스플레이 0.5%+ |
| CPC | 비용/클릭 | 산업별 상이 |
| CVR | 전환/클릭 | 2-5% |
| CPA | 비용/전환 | LTV의 1/3 이하 |
| ROAS | 수익/광고비 | 300%+ |

### 최적화 체크리스트
- [ ] 주간 성과 리뷰
- [ ] 저성과 키워드/타겟 제외
- [ ] 크리에이티브 2주마다 리프레시
- [ ] 입찰 전략 조정
- [ ] 랜딩 페이지 A/B 테스트
- [ ] 경쟁사 광고 모니터링

### 어트리뷰션 설정
- **First-click**: 인지도 캠페인 평가
- **Last-click**: 전환 캠페인 평가 (기본값)
- **Data-driven**: 충분한 데이터 있을 때 권장
