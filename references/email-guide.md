|------|------|----------|
| 오픈율 | 오픈/발송 | 20-25% |
| 클릭률 (CTR) | 클릭/발송 | 2-5% |
| 클릭-오픈률 (CTOR) | 클릭/오픈 | 10-15% |
| 구독취소율 | 취소/발송 | 0.5% 미만 |
| 반송률 | 반송/발송 | 2% 미만 |

## 2. 리스트 빌딩

### 리드 마그넷 유형
| 유형 | 예시 | 적합 산업 |
|------|------|----------|
| 체크리스트 | "SEO 체크리스트 100" | SaaS, 교육 |
| 템플릿 | "마케팅 플랜 템플릿" | B2B, 컨설팅 |
| 가이드/ebook | "A to Z 가이드" | 전문 서비스 |
| 웨비나 | "전문가와 함께하는 Q&A" | B2B, 교육 |
| 무료 체험 | "14일 무료 트라이얼" | SaaS |
| 할인 쿠폰 | "첫 구매 20% 할인" | 이커머스 |
| 퀴즈/진단 | "당신의 마케팅 레벨은?" | B2C, 자기계발 |

### 수집 포인트
1. **웹사이트**: 팝업, 인라인 폼, 종료 의도 팝업
2. **블로그**: 콘텐츠 업그레이드, 사이드바 폼
3. **랜딩페이지**: 전용 리드젠 페이지
4. **소셜미디어**: 바이오 링크, 스토리 CTA
5. **오프라인**: 이벤트, 명함, QR 코드

### 옵트인 베스트 프랙티스
- 더블 옵트인 사용 (품질 > 수량)
- 명확한 가치 제안
- 최소한의 필드 (이메일만으로 시작)
- 개인정보 처리방침 링크

## 3. 이메일 유형

### 자동화 시퀀스

**웰컴 시리즈 (4-7통)**
```
Day 0: 웰컴 + 약속한 리드 마그넷 전달
Day 2: 브랜드 스토리 / 자기소개
Day 4: 핵심 콘텐츠 #1 (가치 제공)
Day 7: 핵심 콘텐츠 #2 (문제 해결)
Day 10: 제품/서비스 소개
Day 14: 케이스 스터디 / 후기
Day 21: 소프트 세일 CTA
```

**온보딩 시리즈 (SaaS)**
```
Day 0: 가입 확인 + 시작 가이드
Day 1: 핵심 기능 튜토리얼
Day 3: 첫 성공 체크인
Day 7: 고급 팁 / 활용 사례
Day 14: 피드백 요청
Day 25: 유료 전환 프로모션 (체험 종료 전)
```

**장바구니 이탈 시리즈 (이커머스)**
```
1시간: 리마인더 (순수 알림)
24시간: 제품 혜택 강조 + 후기
72시간: 한정 할인 제안
```

### 뉴스레터 프레임워크
```
[제목] - 클릭 유도 (호기심, 이익, 긴급성)
[프리헤더] - 제목 보완 (50자 내외)

[인사/훅] - 1-2문장, 관심 유발
[메인 콘텐츠] - 가치 전달 (3-5분 읽기)
[CTA] - 명확한 다음 행동 1개
[PS] - 추가 정보 또는 리마인더
```

## 4. 작성 팁

### 제목 작성 공식
| 공식 | 예시 |
|------|------|
| 숫자 + 이익 | "전환율 3배 높이는 5가지 방법" |
| 질문 | "혹시 이 실수 하고 계신가요?" |
| How-to | "10분 만에 랜딩페이지 만드는 법" |
| 긴급성 | "[마감 임박] 50% 할인 종료" |
| 개인화 | "{이름}님을 위한 특별 제안" |
| 호기심 | "이 한 가지가 모든 걸 바꿨습니다" |

### 본문 작성 원칙
- **F 패턴** 고려: 중요 내용은 왼쪽, 상단에
- **한 문단 = 한 아이디어** (2-3문장)
- **CTA 1개**에 집중 (여러 개 피하기)
- **모바일 최적화**: 짧은 문장, 충분한 여백
- **개인화**: 이름, 행동 기반 콘텐츠

### A/B 테스트 항목
1. 제목 (가장 영향력 큼)
2. 발송 시간/요일
3. CTA 문구/버튼 색상
4. 이미지 유무
5. 콘텐츠 길이

## 5. 전달률 관리

### 스팸 필터 회피
- 스팸 키워드 피하기 (무료!!!, 긴급, $$$)
- 이미지:텍스트 비율 40:60 유지
- 깨진 링크 없이 발송
- 구독취소 링크 명확히 표시
- 정기적인 리스트 정리

### 발신자 평판 관리
- 일관된 발송 빈도 유지
- 비활성 구독자 제거 (6개월 무응답)
- 반송 이메일 즉시 삭제
- SPF, DKIM, DMARC 설정

### 리스트 건강도 체크
```
건강한 리스트:
✓ 오픈율 > 20%
✓ 구독취소율 < 0.5%
✓ 반송률 < 2%
✓ 스팸 신고 < 0.1%
```
