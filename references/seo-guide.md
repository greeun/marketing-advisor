# SEO/콘텐츠 마케팅 가이드

## 1. 테크니컬 SEO 체크리스트

### 필수 설정
- [ ] SSL 인증서 (HTTPS)
- [ ] 모바일 반응형
- [ ] 페이지 로딩 속도 < 3초
- [ ] sitemap.xml 생성 및 제출
- [ ] robots.txt 설정
- [ ] Google Search Console 연동
- [ ] 구조화 데이터 (Schema.org)

### 페이지별 최적화
- [ ] 타이틀 태그 (60자 이내, 키워드 포함)
- [ ] 메타 디스크립션 (160자 이내)
- [ ] H1 태그 (페이지당 1개)
- [ ] 이미지 alt 텍스트
- [ ] 내부 링크 구조
- [ ] URL 구조 (짧고 명확하게)

## 2. 키워드 전략

### 키워드 유형
| 유형 | 의도 | 예시 | 우선순위 |
|------|------|------|---------|
| 트랜잭션 | 구매/가입 | "A 가격", "A 무료 체험" | 높음 |
| 비교 | 결정 | "A vs B", "A 대안" | 높음 |
| 정보 | 학습 | "A란?", "A 사용법" | 중간 |
| 네비게이션 | 브랜드 검색 | "A 로그인" | 낮음 |

### 키워드 리서치 도구
- **무료**: Google Keyword Planner, Ubersuggest, AnswerThePublic
- **유료**: Ahrefs, SEMrush, Moz

## 3. 콘텐츠 전략

### 콘텐츠 유형별 용도
| 유형 | 목적 | 예시 |
|------|------|------|
| 문제 해결 | 상위 퍼널 유입 | "A 오류 해결 방법" |
| 가이드 | 신뢰 구축 | "A 완벽 가이드" |
| 비교 | 전환 유도 | "A vs B 비교" |
| 케이스 스터디 | 소셜 프루프 | "A로 ROI 200% 달성" |
| 템플릿/툴 | 리드 확보 | "무료 A 템플릿" |

### 콘텐츠 작성 체크리스트
- [ ] 타겟 키워드 선정
- [ ] 검색 의도 파악
- [ ] 경쟁 콘텐츠 분석
- [ ] 아웃라인 작성
- [ ] 본문 작성 (1500자 이상)
- [ ] 이미지/도표 추가
- [ ] 내부/외부 링크 추가
- [ ] CTA 삽입

## 4. 링크 빌딩

### 화이트햇 전략
1. **게스트 포스팅** - 관련 블로그 기고
2. **리소스 페이지** - 업계 리소스 목록에 등재
3. **깨진 링크** - 404 페이지 대체 제안
4. **HARO** - 기자 인터뷰 응답
5. **콘텐츠 마케팅** - 링크 유도형 콘텐츠 제작

### 피해야 할 전략
- PBN (Private Blog Network)
- 링크 구매
- 과도한 앵커 텍스트 최적화
- 저품질 디렉토리 등록
